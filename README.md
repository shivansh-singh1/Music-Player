# Music Player

A Spotify-inspired web-based music player built using HTML, CSS, and vanilla JavaScript. This project focuses on creating a clean UI, dynamic song rendering, and real-time audio playback using an external music API.

---

## Features

* Play and pause music with audio controls
* Browse trending and curated song sections
* Search songs dynamically
* Audio progress bar with seek functionality
* Volume control (mute / low / high)
* Responsive design for desktop and mobile devices
* Real-time song preview playback using the iTunes Search API

---

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
* iTunes Search API

## How to Run Locally

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/Music-Player.git
   ```
2. Open the project folder
3. Open `index.html` in your browser

No additional dependencies or setup required.
